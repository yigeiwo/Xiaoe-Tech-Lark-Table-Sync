# 小鹅通-飞书多维表同步工具

一个功能强大的桌面应用程序，用于将小鹅通订单数据自动同步到飞书多维表格。

## ✨ 功能特性

### 核心功能
- 🔄 **自动同步** - 定时自动同步小鹅通订单数据到飞书多维表
- 📊 **字段映射** - 可视化配置字段映射关系，支持嵌套字段
- 🔍 **智能去重** - 自动识别重复订单，支持更新已有记录
- 📅 **时间范围** - 支持自定义时间范围或最近N天数据同步
- 🛡️ **错误重试** - 智能重试机制，确保数据完整性

### 高级特性
- 🎯 **完整字段解析** - 自动解析JSON嵌套字段，包括数组类型
- 📦 **批量处理** - 分批处理大量数据，避免API限流
- 💾 **配置保存** - 自动保存配置，下次启动无需重新配置
- 🖥️ **图形界面** - 友好的GUI界面，无需编程知识

## 📋 系统要求

- Windows 10 或更高版本
- 网络连接（需要访问小鹅通和飞书API）

## 🚀 快速开始

### 1. 下载安装
下载最新的 `小鹅通飞书同步工具.exe` 文件，双击运行即可。

### 2. 获取API密钥

#### 小鹅通API配置
1. 登录小鹅通商家后台
2. 进入「设置」→「开发者设置」
3. 获取以下信息：
   - App ID
   - Client ID
   - Secret Key

#### 飞书多维表配置
1. 创建飞书多维表格
2. 获取多维表格信息：
   - App Token：多维表格URL中的部分，如 `https://feishu.cn/base/XXXXXX` 中的 `XXXXXX`
   - Table ID：数据表的ID，可在URL中查看
3. 创建飞书应用并获取：
   - App ID
   - App Secret
4. 在飞书应用中添加「多维表格」权限

### 3. 配置同步

#### 基础配置
1. 打开工具，进入「基础配置」页面
2. 填写小鹅通API配置（App ID、Client ID、Secret Key）
3. 填写飞书多维表配置（App ID、App Secret、App Token、Table ID）
4. 设置同步参数：
   - 同步最近天数：默认7天
   - 同步间隔：定时同步的时间间隔（分钟）

#### 字段映射
1. 进入「字段映射」页面
2. 点击「获取小鹅通最新字段」自动解析可用字段
3. 点击「一键映射所有字段」自动创建映射关系
4. 或手动添加映射行，选择需要同步的字段

#### 开始同步
1. 进入「运行日志」页面
2. 点击「立即同步一次」进行单次同步
3. 或点击「开始自动同步」开启定时同步

## 📖 使用指南

### 字段映射说明

#### 字段路径格式
- 普通字段：`order_id`
- 嵌套字段：`order_info.order_id`
- 数组字段：`items[].product_name`（自动提取数组中所有元素的该字段）

#### 数据类型处理
- 字符串/数字：直接同步
- 数组：自动转换为JSON字符串
- 嵌套对象：支持多层级嵌套

### 同步策略

#### 新建记录
当订单ID在飞书多维表中不存在时，自动创建新记录。

#### 更新记录
当订单ID已存在时，更新该记录的所有字段。

#### 时间范围
- **最近N天**：同步最近N天的订单数据
- **自定义时间**：指定开始和结束时间范围

### 错误处理

#### 自动重试
- 网络错误：自动重试3次
- Token失效：自动重新获取Token
- 超时错误：指数退避重试

#### 请求限流
- 小鹅通API：每次请求间隔0.5秒
- 飞书API：批量操作每批500条，间隔0.2秒

## 🔧 高级配置

### 配置文件
配置保存在 `sync_config.json` 文件中，包含：
- API密钥配置
- 字段映射关系
- 同步参数设置

### 最大页数限制
默认最多获取100页订单数据（约10,000条），可在代码中修改 `max_pages` 参数。

### 批量处理大小
飞书API批量操作默认每批500条，可在代码中修改 `batch_size` 参数。

## ❓ 常见问题

### Q: 提示"ip not in white list"
**A:** 需要在小鹅通后台将您的IP地址添加到白名单。

### Q: 提示"Token获取失败"
**A:** 检查小鹅通API配置是否正确，确保App ID、Client ID、Secret Key填写无误。

### Q: 字段映射为空
**A:** 确保店铺中有订单数据，工具需要通过实际订单解析字段结构。

### Q: 同步失败
**A:** 检查以下几点：
1. 飞书应用是否有「多维表格」权限
2. App Token和Table ID是否正确
3. 网络连接是否正常
4. 查看日志页面的详细错误信息

### Q: 数据不完整
**A:** 工具会自动解析JSON嵌套字段，包括数组类型。数组字段会以JSON字符串形式存储。
